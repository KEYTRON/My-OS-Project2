ПРОЦЕСС ОТРИСОВКИ (порядок слоёв):

1. graphics_clear() → Очищаем экран цветом DESKTOP_BG_COLOR
2. draw_desktop_wallpaper() → Рисуем фон с градиентом
3. draw_icons() → 4 иконки приложений вверху слева
4. draw_window(окно #1) → Первое окно (500x350)
5. draw_window(окно #2) → Второе окно (400x280)
6. draw_taskbar() → Панель задач с кнопками
7. graphics_flush() → Обновляем видеопамять

СИСТЕМА СОБЫТИЙ (ОБРАБОТКА МЫШИ):

gui_desktop_handle_mouse(x, y, clicked):
  → Проверяет каждую кнопку на панели задач
  → Если мышь над кнопкой: btn->hovered = true
  → Если клик: btn->pressed = true, выводится сообщение
  → Вызывает gui_desktop_render() для обновления
